# 資訊安全管理辦法

第一條　目的
為保護公司資訊資產之機密性、完整性與可用性，特訂定本辦法。

第二條　密碼規定
公司系統帳號密碼長度至少八碼，須包含英文大小寫與數字，並每九十日更換一次。不得共用帳號。

第三條　外部儲存裝置
未經資訊單位核准，不得將公司資料複製至個人 USB 隨身碟或外部雲端硬碟。

第四條　電子郵件
收到可疑郵件或附件時，不得任意點擊連結，應轉寄資訊單位確認。

第五條　離職交接
員工離職時應將經手之系統帳號、資料與權限交接予指定人員，由資訊單位停用其帳號。
