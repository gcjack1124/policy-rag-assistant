# tags 受控詞彙 + 同義詞種子（_ontology）— 路由擴展用

> 用途：路由時把使用者口語擴展成 tags，便宜模型（Haiku）也能命中。
> 「夠不夠」用 **eval 路由命中率**量、非直覺；eval 抓到 miss 才補這裡。
